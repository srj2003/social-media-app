# Ruby on Rails API-Only Social Media App

## Overview

This project is a Ruby on Rails API-only application designed for a social media platform. It implements user authentication using JSON Web Tokens (JWT) and allows users to create and manage posts.

## Features

- User registration with ID, first name, last name, and username.
- User authentication via JWT.
- Creation of posts associated with users.
- Secure API endpoints for user actions.

## Technologies Used

- Ruby on Rails (API mode)
- PostgreSQL (or your preferred database)
- JSON Web Tokens (JWT) for authentication
- RSpec for testing (if applicable)

## Getting Started

### Prerequisites

- Ruby (version 2.7 or higher)
- Rails (version 6.0 or higher)
- PostgreSQL (or your preferred database)
